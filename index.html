Overgrowth Mods

These are all the mods that I made for Overgrowth. Freely available for anyone to use or modify.

Overgrowth is a game made by Wolfire Games. You can preorder the game here.

Landmine

A landmine mod which kills any character stepping onto it. And sends them flying. Source

Bow and Arrow

The bow and arrow combination. Different arrow types have different effects. Complete with quiver which holds up to six arrows. Source

Rocket

The homing rocket will find and kill any character that triggers it. Source

Turret

The turret can keep track of multiple characters coming too close. The turret hotspot has some in-game settings that change it's behaviour. Source

Multiplayer

A two part mod. One part is the client side scripting. The other is the server, made in C++. A collaboration with Surak. Project Organisation

Grenade

A grenade weapon that be be picked up, sheathed and thrown. Works with a timer so once triggered is sure to kill anyone near. Source

Big City

A map made to test the maximum size a level can be in the Phoenix Engine. A small amount of polygons and repeating textures makes it run as good as any other level. Source

Rooftops

A map inspired by the game Mirror's Edge. These skyscrapers are repeated to fill the whole level. Good for parkouring. Source
